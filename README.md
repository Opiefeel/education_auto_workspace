# 📌 FastAPI Microservice and Tests

## 🛠️ Стек технологий
- **📝 Язык**: Python 3.9+
- **⚡ Фреймворк**: FastAPI
- **🚀 Сервер**: Uvicorn
- **🧪 Тестирование**: Pytest


## 🚀 Описание проекта
- ** Тесты хранятся в папке `tests` **
- ** Код микросервиса в файлике main.py **
- ** Модель данных в папке `models` **
